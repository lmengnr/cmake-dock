## Pre-requisites
Install Valgrind
```
sudo apt-get install -y valgrind
```
## Build
```
mkdir build && cd build
cmake ..
make
```

## Run
Use Vagrant to set up a VM

```
vagrant up
vagrant ssh
sudo -i
cd /vagrant/build/
```


## Use Valgrind:
```
valgrind ./container run
```
